# Compass — BPI Product Assessment
Built as part of a product design and prototyping assessment for BPI Labs.

Compass is a prototype tool that turns a campaign brief into a ranked list of media targets using policy elite engagement data.

## What it does
- Accepts a campaign brief
- Applies filters based on available dataset fields
- Returns a ranked top 25 list of targets
- Generates AI-powered reasoning for each recommendation
- Allows users to select and export targets

## Tech Stack
- Next.js / Vercel
- Claude API
- CSV dataset

## Live Demo
https://bpi-compass.vercel.app/

## Notes
This is a prototype built for a product assessment. It focuses on core workflow and decision-making, not production-level features like authentication or persistence.
